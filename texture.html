Distortion is the alteration of an object's appearance to create a new, unique appearance. There are many types of distortion. Some types include abstraction, manipulation, and exaggeration. Abstraction is the process of distorting and simplifying the form of an object to create an abstract appearance. Manipulation is the alteration of the visual elements of an object or work to create new, unexpected objects. This can be intentional to get a viewer to see a certain perspective. Exaggeration is when certain features of an object are amplified to intentionally emphasized. Distortion is done to invoke certain emotions and thoughts.
